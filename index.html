Help:
This site synchronizes playback of multiple videos and supports playback in multiple tabs. It probably only works in recent versions of Chrome and Firefox.
Each browser tab takes part in a session and each session contains videos whose playback is synchronized. Each video has a starting time which is used for synchronization and each tab can show or hide videos individually.
If you are playing videos with shared audio for example from a Let's play by multiple people you probably want to mute all but one video.
Points in time for starting times can be entered in seconds (for example 30.5) or in the format hours:minutes:seconds (for example 0:1:1.5).
Everything works in multiple tabs: You can have two synchronized videos playing in two browser windows which is useful for making both fullscreen on two monitors.

Press H to toggle the user interface. The playback controls auto fade out in this mode and can be dragged.
Press Space to pause / unpause all videos. Note that for key presses you might need remove focus from the youtube players by clicking somewhere else in the document first.

The source code is here: https://github.com/e00E/Youtube-Synchronizer.
